makemovie

PURPOSE

[3D] Used to convert a stack of T images into a movie.

SYNOPSIS

function M = makemovie( IS )

DESCRIPTION 

 [3D] Used to convert a stack of T images into a movie.  

 To display same data statically use montage.

 INPUTS
   IS              - MxNxT or MxNx1xT or MxNx3xT array of movies.

 OUTPUTS
   M               - resulting movie

 EXAMPLE
   load( 'images.mat' );
   M = makemovie( videos(:,:,:,1) );
   movie( M );
